Our Projects

Our Projects

Our team has successfully delivered a variety of projects across different industries, focusing on cutting-edge technologies, cloud services, and complex financial systems. Here are some of our notable works:

Intelligent Office

Financial Advisers Platform

The Intelligent Office is a comprehensive platform used by financial advisers to service their customers effectively. This platform integrates various modules, including client management, transaction handling, and reporting. The tech stack used includes .NET Core, Microservices, C#, EF Core, Dapper, Nhibernate, MVC, Angular, and React, allowing for a scalable and efficient solution.

I worked on this project from the very beginning, participating in the requirements gathering stage, where we collaborated with stakeholders to understand the client's needs. My role involved creating user stories, refining requirements, and ensuring that the project followed the Agile methodology throughout its lifecycle. I also ensured that each module was delivered as per the defined scope and user stories.

AWS was used for cloud infrastructure, and I leveraged services like AWS Lambda, EC2, and S3 to deploy and manage various microservices, enhancing the platform's scalability and security. Developers worked in sprints to develop features, implement APIs, and conduct extensive testing to ensure product quality.

ACATS - Account Transfer and Money Movement

Financial Transfer System

The ACATS (Automated Customer Account Transfer Service) project focused on account transfer and money movement from one account to another in the United States. The application was built with the same tech stack as the Intelligent Office, but this time, we integrated Azure cloud services.

My responsibilities included working with MQ systems to place messages, implementing the USA guidelines for sending SI (Settlement Instructions) and TI (Transaction Instructions) records, and ensuring that the system was capable of handling large-scale transactions securely and efficiently. Azure services like Azure Storage, Azure Functions, and Azure Logic Apps were used to integrate and automate workflows, ensuring scalability and availability.

Imports Platform

USA Imports Trading System

The Imports Platform enables the importation of goods into the USA, adhering to the CBP (Customs and Border Protection) guidelines. I worked on migrating the old platform to a new .NET Core application and integrating Azure cloud services. Azure's DevOps, AKS (Azure Kubernetes Service), and Azure SQL Database played a significant role in the transition, improving scalability and reliability.

This project required ensuring compliance with CBP guidelines and automating the flow of information for trading imports. Developers worked with legacy systems to migrate data and implemented modern solutions to handle future scalability challenges.

ISF - Importer Security Filing

Security Filing System for Importers

The ISF (Importer Security Filing) project involves implementing the security filing process for importers in the United States. This system ensures that necessary data is submitted to CBP before cargo is loaded onto vessels. I worked on upgrading the platform to .NET Core and integrated it with Azure for cloud services.

Azure Cloud provided solutions like Azure Functions for processing incoming data and Azure Active Directory for secure access. Developers focused on automating security filing processes while ensuring that data integrity and compliance with the latest regulations were maintained. We also implemented real-time data tracking to enhance reporting accuracy.
